# Molecular Dynamics (Velocity-Verlet)

Small, educational implementation of a velocity-Verlet integrator using a
pairwise Lennard-Jones potential.

Files
- `molecular_dynamics.py`: integrator, forces, energy, example run.
- `analysis.md`: complexity and correctness notes.
- `tests/`: unit tests.
1. **Molecular Dynamics (Verlet Algorithm)** - Atomic trajectory integration
2. **Metropolis Monte Carlo** - Statistical sampling
3. **Nudged Elastic Band** - Reaction pathway finding
4. **Kinetic Monte Carlo** - Long-timescale dynamics
5. **Embedded Atom Method** - Metallic interatomic potentials
